# 🧠 Memory & Virtual Memory Management Simulator

A full-stack Operating Systems project that simulates **Memory
Allocation** and **Page Replacement Algorithms** with an interactive
GUI.

------------------------------------------------------------------------

## 🚀 Overview

We developed a Memory & Virtual Memory Management Simulator with a
graphical user interface, where core algorithms are implemented in C and
integrated via a Python Flask backend.

This project demonstrates core OS concepts by combining:

-   ⚙️ C Language → Core algorithm implementation
-   🌐 Python Flask → Backend bridge
-   🎨 HTML, CSS, JavaScript → Interactive frontend

------------------------------------------------------------------------

## 🎯 Features

### Memory Allocation

-   First Fit
-   Best Fit
-   Worst Fit
-   Fragmentation Analysis

### Page Replacement

-   FIFO (First In First Out)
-   LRU (Least Recently Used)
-   Optimal Algorithm

------------------------------------------------------------------------

## 🏗️ Architecture

Frontend → Flask API → C Engine

------------------------------------------------------------------------

## ⚙️ Setup

1.  Clone repo

2.  Compile C: gcc backend/engine.c -o backend/engine

3.  Run server: python3 server.py

4.  Open frontend
